# How to Install Git


### Mac OSX

It should came installed with the OS

To check it: 

 ```shell
 	> git --version
 	git version 2.x.x ....
 ```	

In case it is not installed:

Make sure that **brew** is installed and run:

```shell
brew install git
```

Now check that is there.
